TESS Visibility Tool

Enter a sky coordinate below to check if it falls inside the TESS field of view.


Alternatively, upload a file.

API Documentation

/in-tess-fov?pos={POS} [example]
Returns "yes" if the sky coordinate specified by POS is observeable by TESS in Cycle 1, "no" otherwise. POS must be a string specifying a sky coordinate in decimal or sexagesimal format, e.g. "270.0 -28.0", "18h00m00s -28d00m00s", or "18:00:00 -28:00:00". You can supply multiple coordinates separated by commas.

/tessvis.png?pos={POS} [example]
Returns a plot showing the Cycle 1 TESS visible region. Optionally, the coordinate specified by POS can be marked. See above for the formatting of POS.
